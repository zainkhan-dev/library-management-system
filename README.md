# 📚 Library Management System

A simple command-line **Library Management System** built with **Python**. This project demonstrates the use of Object-Oriented Programming (OOP), file handling, exception handling, and modules to create a functional application that manages library books.

---

## 🚀 Features

* ➕ Add a new book
* ❌ Delete a book
* 📖 Borrow a book
* 🔄 Return a book
* 🔍 Search for a book
* 📋 View all books
* 💾 Save library data automatically using JSON

---

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* JSON File Handling
* Exception Handling
* Python Modules

---

## 📂 Project Structure

```text
LIBRARY MANAGEMENT SYSTEM/
│
├── main.py              # Main program
├── library.py           # Library class and operations
├── book.py              # Book class
├── file_handler.py      # Save and load data
├── library_data.json            # Stores library data
└── README.md
```

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/zainkhan-dev/library-management-system
```

### 2. Navigate to the project folder

```bash
cd LIBRARY MANAGEMENT SYSTEM
```

### 3. Run the application

```bash
python main.py
```
